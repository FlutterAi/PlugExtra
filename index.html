PlugExtra

Enhances plug.dj with new features!

Download this project as a .zip file Download this project as a tar.gz file

Add this address to your bookmarks: PlugExtra

Click it in a plug.dj room to start PlugExtra.

Current features:

  • Automatic woot or waitlist join. Can be turned off.
  • Shows a log of playing tracks and their votes and curates. The log can be hidden.
