PlugExtra Version 1.2.5

Enhances plug.dj with new features!

Download this project as a .zip file Download this project as a tar.gz file

Add this address to your bookmarks for the normal version: PlugExtra
Dev version: PlugExtraDev

Click the bookmark in a plug.dj room to start PlugExtra.

Current features:

  • Automatic woot or waitlist join. Can be turned off.
  • Shows a log of playing tracks and their votes and curates. The log can be hidden.
  • Lists all online users and users in the waitlist on small space.
  • Awaybot that replies with a specified message when you're being mentioned.
  • Displays a message at which place a user left the waitlist or dj booth.
  • Shows how many users left the waitlist or dj booth during a song.

Tested in Mozilla Firefox and Google Chrome.
